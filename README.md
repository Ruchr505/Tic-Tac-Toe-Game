# Tic-Tac-Toe Game
### 1. ติดตั้ง Dependencies

- Backend:
  - เข้าโฟลเดอร์ backend
  - รัน `npm install`
- Frontend:
  - เข้าโฟลเดอร์ frontend
  - รัน `npm install`

### 2. รันโปรเจกต์

- Backend:
  - เข้าโฟลเดอร์ backend
  - รัน `npm run start`
- Frontend:
  - เข้าโฟลเดอร์ frontend
  - รัน `npm run dev`
- Both
  - รัน `npm run dev` เมื่อเข้าโปรเจค

### 3. เปิดใช้งาน

- เข้าเว็บที่ `http://localhost:5173`

## Environment Variables

- ตั้งค่าไฟล์ `.env` ใน backend และ frontend ตามตัวอย่างที่มี

---
**หมายเหตุ:** เปิด docker ก่อนรันโปรแกรม
